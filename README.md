# Interactive Cubic Bézier Curve

An interactive web app to visualize and manipulate a **cubic Bézier curve** using HTML, CSS, and JavaScript. Move the control points and see how the curve updates in real-time.

---

## Features

- Visualize a **cubic Bézier curve** with 4 points:
  - Start point (P0)
  - Control points (P1, P2)
  - End point (P3)
- Drag control points to dynamically update the curve.
- Real-time rendering using **HTML5 Canvas**.
- Lightweight and easy to understand for learning purposes.
